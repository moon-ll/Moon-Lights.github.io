# 如何添加视频内容

想把录好的 `.mp4` 视频放到网站上展示？非常简单。

## 步骤

1. **把你的视频文件复制到 `content/` 目录下**

2. **打开 `content/index.json`，添加一条记录**：

```json
{
  "id": "my-video",
  "title": "我的视频标题",
  "type": "video",
  "file": "my-video.mp4",
  "date": "2026-04-01",
  "summary": "这是一段关于...的视频",
  "tags": ["视频", "讲解"]
}
```

3. **刷新 `content.html` 页面**，左侧列表就会出现新条目，点击即可播放。

## 视频压缩建议

- 如果你的视频很大，建议先用工具压缩一下（比如 HandBrake、FFmpeg）
- 推荐分辨率：1920×1080 或 1280×720
- 推荐格式：H.264 编码的 `.mp4`，兼容性最好

## 提示

视频是直接在浏览器里播放的，没有上传到第三方平台，所以完全离线可用。
