# 欢迎使用内容中心

这里是你整理和分享内容的地方。你可以把 Markdown 文章、MP4 视频或者 HTML 演示页面都放到 `content/` 目录下，网站会自动读取并展示。

## 怎么添加内容？

非常简单，只需要两步：

1. **准备文件**
   - 写文章：新建 `.md` 文件，用 Markdown 语法书写
   - 放视频：把 `.mp4` 文件放到 `content/` 目录
   - 做演示：新建 `.html` 文件，写你的动画或交互 Demo

2. **修改索引**
   打开 `content/index.json`，仿照现有格式添加一条记录：

```json
{
  "id": "my-post",
  "title": "我的文章标题",
  "type": "article",
  "file": "my-post.md",
  "date": "2026-04-01",
  "summary": "这是一段摘要，会显示在列表里。",
  "tags": ["标签1", "标签2"]
}
```

## 支持的类型

| 类型 | 说明 | file 后缀 |
|------|------|-----------|
| `article` | Markdown 文章 | `.md` |
| `video` | 本地视频 | `.mp4` |
| `demo` | HTML 演示/动画 | `.html` |

刷新 `content.html` 页面，新内容就会出现在左侧列表中。

## 写作提示

- 你可以插入图片，只要路径写相对 `content.html` 的位置即可
- 代码块会自动保留等宽字体样式
- 标签会显示在标题下方，方便分类浏览
